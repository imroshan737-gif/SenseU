# 🌌 NeuroAura – AI-Powered Mental Wellness System

## 🧠 Overview
NeuroAura is an **AI-driven mental wellness platform** that helps users **monitor, analyze, and reduce stress** through intelligent insights, scientific assessments, and personalized interventions.

It combines **predictive analytics, adaptive AI, and immersive UI** to deliver a **data-driven and engaging mental health experience**.

---

## 🚀 Key Features

### 🧾 Smart Onboarding & Assessment
- Guided **sign-up/login with permissions**
- Initial **6-question adaptive assessment**
- Input types:
  - Sliders (1–10)
  - Short/long responses
  - Select options
- Includes **transparent scoring explanation**

---

### 📊 AI-Based Stress Analysis Engine
- Multi-factor stress calculation using:
  - Questionnaire data
  - Behavioral patterns
  - Sleep inputs
  - App interaction
- Outputs:
  - **Stress Score**
  - **Confidence Level**
  - **Key insights**
---

### 🤖 Aurora – Intelligent AI Assistant
- Context-aware **AI wellness assistant**
- Supports:
  - Conversations
  - Image analysis
  - Personalized guidance
- Learns from **user behavior and history**

---

### 📈 Predictive Mental Health Analytics
- Predicts **future stress trends**
- Identifies:
  - Daily stress patterns
  - Sleep-related impacts
- Provides **preventive suggestions**

---

### 🧬 Adaptive Personalization Engine
- Learns user-specific responses to interventions
- Tracks **stress reduction effectiveness**
- Recommends **best-suited activities**

---

### 📉 Live Vitals Dashboard
- Real-time metrics:
  - **Stress**
  - **Focus**
  - **Energy**
- Interactive and immersive UI
